gTalk Syntax Highlighter Options

Syntax Highlighter can use a Baysian algorithm to automatically determine which chat messages should be highlighted.

gTalk Syntax can send your chat messages data back to its server to improve its machine learning. Any data sent will not in any way be treated as private information, as those words are used unencrypted for client-side baysian intelligence.

You can also use dev mode, which allows a custom server endpoint.
